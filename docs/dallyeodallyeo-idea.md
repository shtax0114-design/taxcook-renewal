# 달려달려 앱 아이디어 정리

## 한 줄 소개

달려달려는 배달 라이더가 오늘 번 돈과 실제 남은 돈을 한눈에 정리하는 무료 수익 관리 앱이다.

## 핵심 문제

배달 라이더는 여러 플랫폼을 오가며 일하지만, 실제 수익을 한곳에서 보기 어렵다.

- 오늘 총수입은 알지만 실제 순수익은 바로 계산하기 어렵다.
- 배달 건수, 운행 시간, 주행거리, 지출을 따로 관리해야 한다.
- 배민, 쿠팡이츠, 요기요 등 플랫폼별 수익 효율을 비교하기 어렵다.
- 종합소득세 신고를 앞두고 월별 수입과 경비 정리가 번거롭다.

## 핵심 방향

처음부터 배달 플랫폼 API 연동을 목표로 하기보다, 라이더가 직접 입력하는 간단한 수익 계산기에서 시작한다.

중요한 것은 복잡한 기능이 아니라 매일 20초 안에 입력할 수 있는 편한 사용성이다.

## MVP 기능

- 오늘 기록 추가
- 날짜 선택
- 플랫폼 선택
  - 배민
  - 쿠팡이츠
  - 요기요
  - 기타
- 총수입 입력
- 운행 시간 입력
- 배달 건수 입력
- 주행거리 입력
- 지출 입력
  - 주유
  - 식비
  - 정비
  - 보험
  - 렌탈
  - 기타
- 메모 입력

## 자동 계산 항목

- 오늘 총수입
- 오늘 총지출
- 오늘 순수익
- 시간당 수익
- 건당 수익
- km당 수익
- 주간 합계
- 월간 합계
- 플랫폼별 수익 비교

## 화면 방향

라이더가 이동 중이나 휴식 중에 빠르게 입력할 수 있어야 한다.

- 버튼은 크고 시원하게
- 입력칸은 최소화
- 숫자 입력은 키패드가 바로 뜨게
- 플랫폼 선택은 카드형 큰 버튼
- 저장 버튼은 눈에 띄게 크게
- 결과는 한눈에 보이게

첫 화면 예시:

```text
오늘 얼마 벌었나요?

[배민] [쿠팡이츠] [요기요] [기타]

총수입
운행시간
배달건수
주행거리
지출

[오늘 기록 저장]

오늘 순수익
시간당 수익
이번 달 누적
```

## 사용자 가치

라이더는 매일 수입을 기록하면서 실제 남은 돈을 바로 확인할 수 있다.

예시:

```text
오늘 매출: 120,000원
기름값: 15,000원
식비/커피: 8,000원
보험/렌탈 일할 계산: 12,000원
실제 순수익: 85,000원
시간당 순수익: 14,100원
```

## 수익 모델

앱은 무료로 제공한다.

광고는 일반 구글 광고처럼 지저분하게 붙이지 않고, 라이더에게 필요한 혜택형 광고와 제휴 중심으로 구성한다.

광고/제휴 후보:

- 오토바이 정비소
- 타이어 교체
- 엔진오일 교체
- 배달 가방
- 우비, 방한용품
- 헬멧, 장갑
- 블랙박스
- 이륜차 보험
- 오토바이 리스/렌탈
- 휴대폰 거치대
- 보조배터리
- 라이더 쉼터/카페
- 세금 신고/장부 정리
- 건강검진
- 산재/보험 상담

광고는 배너보다 혜택처럼 보이게 한다.

예시:

```text
이번 달 라이더 전용 혜택
엔진오일 교체 10% 할인
배달가방 방수커버 특가
종소세 신고 준비 체크리스트 무료
```

## 택스쿡과의 연결 가능성

달려달려는 라이더 수익 기록 앱으로 시작하지만, 자연스럽게 세금 신고 준비와 연결될 수 있다.

- 월별 수입 정리
- 경비 항목 정리
- 종합소득세 신고용 자료 다운로드
- 예상 세금 계산
- 택스쿡 종합소득세 신고 연결

## 확장 아이디어

- 월간 리포트 PDF
- 세금 신고용 엑셀 다운로드
- 플랫폼별 수익 그래프
- 시간대별 수익 효율 분석
- 지역별 메모 기능
- 오늘 달리기 좋은 시간 추천
- 라이더 커뮤니티 제보
- 정비/보험/용품 할인 쿠폰
- 프리미엄 리포트 기능

## 포지셔닝

달려달려는 라이더의 수익을 정리해주고, 라이더에게 필요한 혜택만 보여주는 무료 앱이다.

## 후보 문구

- 오늘도 달렸다면, 남은 돈은 알아야죠
- 오늘 번 돈, 진짜 남은 돈까지
- 달린 만큼 남기는 라이더 수익 관리
- 배달 수익, 이제 달려달려에 정리하세요
- 라이더를 위한 가장 쉬운 수익 계산기

## 초기 결론

처음부터 배달 플랫폼 API 통합을 목표로 하면 법적, 기술적 장벽이 높다.

따라서 1차 버전은 라이더가 직접 입력하는 무료 수익 관리 앱으로 시작한다. 사용자가 쌓이면 광고/제휴, 세금 신고 연결, 리포트 기능으로 확장한다.
